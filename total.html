🧮 Relatório Oficial – Regulados por Área

Estabelecimentos de alta complexidade distribuídos por área de atuação.

1) Identificação e Escopo
Este relatório contabiliza a quantidade de Regulados classificados como Alta Complexidade por Área de Atuação, a partir de arquivos JSON hospedados em repositório institucional (GitHub Pages).

Premissa institucional: todos os registros listados no index_regulados.json são considerados ativos (logo, não há filtro de status).
2) Fonte de Dados e Regras de Cálculo (Auditável)
2.1 Fonte primária: data/index_regulados.json (estrutura { meta, dados[] }, com dados[].codigo).
2.2 Fonte detalhada: reg/00..43/<codigo5>.json, onde a pasta é formada pelos dois primeiros dígitos do código (ex.: 13022 → reg/13/13022.json).

2.3 Critério “Alta Complexidade”: considera somente itens em atividades[] cujo campo complexidade seja exatamente “ALTA”.
Se o regulado não possuir nenhuma atividade com “ALTA”, é descartado do universo deste relatório.

2.4 Regra de contagem única por regulado: cada regulado é contabilizado apenas uma vez, sendo classificado em uma única área.
2.5 Prioridade de área (ordem fixa): Medicamentos → Produtos para Saúde → Saneantes → Serviços de Saúde → Indústria de Alimentos.
2.6 Fallback: quando nenhuma atividade “ALTA” estiver em uma das áreas acima, o regulado é atribuído a qualquer uma das áreas remanescentes presentes nos seus CNAEs “ALTA” (seleção aleatória).
